# TailornadoFrontend

## Описание проекта
Приложение (интернет магазин одежды) на flutter расчитанное как на мобильные устройства так для пк.

## Основной функционал
- Авторизация: Вход и регистрация пользователей. Работает через JWT-токены, которые сохраняются в памяти устройства.
- Роли: Логика приложения меняется в зависимости от роли (покупатель или продавец). Продавец может добавлять новые товары через специальный диалог.
- Каталог: Список товаров с подгрузкой данных. Есть фильтрация по брендам и диапазону цен.
- Чат: Реализован на WebSocket. Есть два режима: обычный пользователь пишет в поддержку, а менеджер может видеть список ожидающих пользователей и отвечать на вопросы.
- Локализация: Поддержка двух языков (русский и английский) через easy_localization.
- Темы: Темная и светлая темы, переключаются в один клик.

## Стек используемых технологий
- Flutter — основной фреймворк.
- Provider — управление состоянием приложения.
- http & web_socket_channel — для работы с API и чатом.
- Shared Preferences — хранение токена и настроек.
- Docker — для быстрой сборки и запуска

## Структура проекта
- lib/data — модели данных, провайдеры (логика) и сервисы для запросов к бэкенду.
- lib/presentation — визуальная часть: страницы (pages), виджеты (widgets) и темы оформления.
- lib/core — вспомогательные утилиты (логгер).
- assets/translations — файлы переводов.
